# bpmn.js流程设计器组件
```
1、调整设计器UI布局，基于vue-elementui美化属性面板
2、支持设置任务变量、表达式、分支条件等，满足90%以上的业务需求
```
## 安装运行
```
yarn install
yarn serve
yarn build
```
